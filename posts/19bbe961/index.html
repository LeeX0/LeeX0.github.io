微醺日记——清单

微醺日记——清单

持续更新

Whisky

Whisky Sour（威士忌酸）

Daiquiri大吉利——烈酒+柑橘酸+糖浆

属于酸＋甜结构。更改基酒核心即可（威士忌酸，白兰地酸，金酒酸，伏特加酸等）参考

40ml Whisky + 15ml 蜂蜜糖浆 + 挤半个柠檬 ，满冰Shake壶Shake it，倒入Martini杯，挤柠檬皮油并装饰
tips：喜欢清爽感可以再加气泡水

关于糖浆

单糖浆：水煮沸（冷水也可以）加入等量白砂糖，完全溶解冷藏

蜂蜜糖浆：将半杯水和半杯蜂蜜加入锅中，一边中火慢煮，一边搅拌至混合，等水煮干后滤入瓶中，用盖子密封，可在冰箱中保存1个月（用的蜂蜜越多，糖浆就越稠，味道就越浓）

别的各种品类糖浆可以直接购买成品

Jim Beam Highball

40ml Bourbon Whisky + 120ml 苏打水，满冰Highball杯，可加适量柠檬与薄荷等

Jim Beam Ginger Highball

40ml Bourbon Whisky + 5-10ml 柠檬汁 + 100ml 姜汁汽水，满冰Highball杯

Jim Beam Cola Highball

40ml Bourbon Whisky + 5-10ml 柠檬汁 + 适量可乐

Gin

Gin Tonic（金汤力）

40ml 金酒 + 120ml 汤力水，满冰Highball杯，吧勺上下搅拌勾匀，喷挤柠檬皮油

Vodka

Vodka Lemon

40ml Vodka + 挤半个柠檬切角，大块冰块，喷挤柠檬皮油

Vodka Soda

Vodka Lemon基础上换成Highball杯，苏打水加满

Vodka Lemon Drop

50ml Vodka + 15ml糖浆 + 挤半个柠檬，满冰Shake壶Shake it，倒入shot杯，搭配粘上糖粉的柠檬角（喝完酒吃）

Rum

Mojito（莫吉托）

1shot朗姆酒 + 2匙糖 + 青柠薄荷 + 气泡水，冰块与碎冰大半杯

其他

百利甜奶茶

1shot 百利甜 + 120ml 奶茶，满冰Highball杯，打发奶泡倒在最上面
